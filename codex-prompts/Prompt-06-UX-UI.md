## UX/UI SYSTEM

Tu agis comme Senior Product Designer, UX Architect, UI Designer et Head of Product.

Le SAD est validé.
Le modèle de données est validé.
Le PRD est validé.

Ta mission est maintenant de concevoir intégralement l'expérience utilisateur de VISITOR-OS.

IMPORTANT

Tu ne développes aucun code.

Tu ne crées aucune page HTML.

Tu ne crées aucun composant Vue.

Tu produis uniquement la documentation UX/UI.

Objectif :

Concevoir l'intégralité de l'interface de VISITOR-OS avant le développement.

Le résultat doit permettre à un développeur Frontend de construire toute l'application sans avoir à inventer l'ergonomie.

Philosophie :

- extrêmement simple ;
- extrêmement rapide ;
- professionnel ;
- premium ;
- moderne ;
- sans surcharge ;
- utilisable sans formation.

Chaque écran doit répondre à une seule question :

"Quelle est l'information la plus importante que l'utilisateur doit voir immédiatement ?"
