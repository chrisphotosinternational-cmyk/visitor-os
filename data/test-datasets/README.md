# VISITOR-OS Beta Test Datasets

These CSV files are intentionally dirty and are used for beta validation.

Files:

- `prospects_small_dirty.csv`: 100 rows
- `prospects_medium_dirty.csv`: 5,000 rows
- `prospects_large_dirty.csv`: 50,000 rows
- `prospects_extreme_dirty.csv`: 100,000 rows

They contain duplicates, invalid emails, malformed phones, missing cities, broken URLs, incomplete profiles, special characters, accents, empty rows, unknown columns, long values and partially inconsistent data.

These datasets must never be treated as production data.
