# OpenAI Provider

OpenAI is available through the AI Provider Engine, not through the Decision Engine.

## Environment

```bash
OPENAI_API_KEY=
```

If the key is empty, VISITOR-OS falls back to the mock provider automatically.

## Default Model

The recommended initial model is:

```text
gpt-4o-mini
```

The model remains configurable per organization.

## Safety

The OpenAI provider:

- receives a system prompt generated by the Prompt Builder;
- receives recent conversation context;
- returns reply, latency, tokens and estimated cost;
- does not contain business rules;
- does not decide escalation alone.

Business rules, FAQ and knowledge base remain local and are evaluated before any AI call.
