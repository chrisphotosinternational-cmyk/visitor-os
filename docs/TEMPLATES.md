# Chatbot Studio Templates

Templates accelerate chatbot creation while remaining editable.

Initial templates:

- Chambre d'hotes
- Photographe
- Decoration
- SEO
- Restaurant
- Coach

Each template can create:

- intentions;
- basic knowledge items;
- goals;
- personality defaults;
- CTA labels.

Templates are starting points only. They do not publish content automatically. Generated content starts as draft knowledge and must be reviewed before publication.

## Principles

- A template must be understandable by a non-technical administrator.
- A template must never hard-code customer-specific facts.
- A template must produce editable draft content.
- A template must respect the existing Knowledge Engine model.
