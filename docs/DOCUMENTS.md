# Knowledge Documents

Each document contains:

- id
- organization
- site
- title
- description
- category
- type
- language
- version
- import date
- size
- hash
- status
- tags
- author
- source

## Supported RC1 Types

- Markdown
- TXT
- HTML
- PDF
- DOCX
- CSV
- JSON
- existing FAQ
- existing Knowledge Base

RC1 expects extracted text content. Heavy binary parsing is intentionally deferred.
