# VISITOR-OS User Guide - v1.0.0-RC1

## Login

Open the admin URL, enter your email and password, then confirm. If the session expires, reconnect.

## Prospects

Use `/prospects` to search, filter, create, edit, import and export prospects. Scores are explainable indicators, not automatic decisions.

## Follow-ups

Use `/follow-ups` to see overdue and upcoming manual actions. VISITOR-OS does not send outreach automatically.

## Messages

Use `/message-templates` to prepare reusable messages. Rendering a message only creates text to copy. Sending remains manual.

## Enrichment

Use public enrichment only for public pages. VISITOR-OS must not bypass login, captcha, paywall or private areas.

## Pipeline

Use `/pipeline` to move prospects through commercial stages and keep conversion metrics current.
