# VISITOR-OS FAQ - v1.0.0-RC1

## Is VISITOR-OS a chatbot?

No. VISITOR-OS is a Customer Intelligence Platform. The widget and AI are modules, not the full product.

## Can it run on OVH Web mutualise?

No for the backend. OVH Web mutualise can host static pages, but VISITOR-OS backend needs a Node.js-compatible platform.

## Does VISITOR-OS send prospect messages automatically?

No. Outreach remains manual in RC1. Templates can be rendered, copied and historized.

## Does enrichment access private data?

No. Only public pages are allowed. No login bypass, captcha solving, paywall bypass or private scraping.

## Is PostgreSQL mandatory?

The backend can boot without it for health checks, but production functionality requires PostgreSQL.
