# Tags

Sprint 7 adds automatic and manual CRM tags.

## Default Tags

- Reservation
- Tarif
- Disponibilite
- Day Use
- Week-end
- Teletravail
- Festival
- Parking
- Petit-dejeuner
- Acces
- Decoration murale
- Shooting photo
- Studio photo
- Urgent
- Autre

## Scope

Tags are organization-scoped and can optionally be associated with a site.

They can be attached to:

- prospects;
- conversations.

## Sources

- `automatic`: detected from visitor messages;
- `manual`: added by an admin user.
