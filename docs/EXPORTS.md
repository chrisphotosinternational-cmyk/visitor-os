# Exports

Sprint 7 adds filtered CRM exports.

## Formats

- CSV;
- XLSX-compatible XML spreadsheet.

## Filters

Exports can be filtered by:

- organization;
- site;
- status;
- score;
- tag;
- period.

## Columns

- Date
- Organisation
- Site
- Prenom
- Nom
- Email
- Telephone
- Statut
- Score
- Tags
- Derniere question
- Derniere reponse
- Notes
- Relance
- Source

## Security

Exports require the `data:export` permission and must respect organization isolation.
