# Reasoning Lab

Le Reasoning Lab permet a un administrateur de tester une question avant publication ou correction de connaissances.

## Acces

Dans l'administration :

```text
Knowledge Engine -> Chatbot -> Reasoning Lab
```

## Resultats affiches

Le test retourne :

- reponse generee ;
- intention detectee ;
- score de confiance ;
- connaissance utilisee ;
- objectif applique ;
- personnalite appliquee ;
- next best action ;
- trace technique.

## Usage recommande

Utiliser le Reasoning Lab pour verifier :

- une nouvelle base de connaissances ;
- un objectif de reservation ou devis ;
- une personnalite plus rassurante ;
- les questions qui declenchent un fallback ;
- les cas ou le chatbot doit capturer un lead.
