# Lead Readiness

Le lead readiness est un score de 0 a 100 qui estime si une conversation merite une capture de coordonnees ou une conversion CRM.

## Regles V1

- demande de tarif : +15 ;
- demande de disponibilite : +20 ;
- demande de reservation ou devis : +25 ;
- email ou telephone donne : +30 ;
- urgence mentionnee : +15 ;
- demande simple d'information : +5 ;
- conversation courte sans intention commerciale : +0.

## Seuils

- `> 60` : capture lead recommandee ;
- `> 80` : conversion CRM recommandee ;
- `< 40` : reponse simple suffisante dans la majorite des cas.

## Conformite

Le score ne declenche aucun envoi automatique. Il aide uniquement l'administrateur et le widget a choisir une action douce.
