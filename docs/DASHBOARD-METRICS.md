# Dashboard Metrics

The admin analytics dashboard exposes the following metrics.

## KPI Cards

- Conversations
- Prospects created
- Visitor to prospect conversion rate
- Average score
- Hot prospects
- Follow-ups today
- Follow-ups overdue
- Fallback rate
- Human escalation rate
- Estimated AI cost
- Notifications sent
- Important errors

## Charts

V1 uses lightweight CSS bars instead of a charting library.

- Conversations per day
- Prospects per day

## Tables

- Site performance
- Frequent tags
- Response sources

## Response Sources

Sources come from Decision Engine events:

- FAQ
- knowledge base
- AI
- fallback
- human escalation

## Alerts

Important alert counters include overdue follow-ups and failed/system notifications.
